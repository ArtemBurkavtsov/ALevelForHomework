SQL - Structured Query Language. NOSQL это Not Only SQL ( то есть, не только SQL ).
Они представляют собой базой управления базами данных, но они отличается подходами и реализацией
сохранения и обработки данных.

Плюсы и минусы обоих:

Плюсы: SQL: Строгая конститетность , благодаря трансзакциям. Эффективно работает с сложными
            запросами
       NOSQL: Гибкость схем, быстрее работает чем SQL

Минусы: SQL: Нет той гибкости, которая есть у NOSQL
        NOSQL: Нет той конститености что есть  SQL

Где применяется:
SQL: В финансовых структорах, возможно в комерции
NOSQL: Системы анализа данных, соц.сети и игровые платформы
